# Maya Reference Easy Import (參考影片快速匯入工具)

這是一個專為 Maya 動畫師設計的流程優化工具，解決製作動畫時導入參考影片的繁瑣設定。  
只需一鍵點擊，工具便會自動將影片轉換為序列圖檔、建立專屬控制器並完美綁定於攝影機上。  
適合用於觀察動作節奏、重心變化、臉部表情，或是在 Layout 階段讓導演快速理解畫面構想。  

---

## 🛠️ 安裝方法 (Installation)

1. 下載專案後，將內部的 `Reference_easy_import` 資料夾完整放入 Maya 的腳本目錄中：
   * `Documents\maya\scripts\`
2. 打開 Maya 軟體。
3. 將最外層的 `Install.mel` 檔案，直接用滑鼠拖曳進 Maya 的 3D 視圖 (Viewport) 中放開。
4. 您的 `Custom` 工具列 (Shelf) 上會自動生成專屬工具按鈕，點選即可使用！

*(備註：或者您也可以在完成步驟 1 後，於 Maya 的 Script Editor 中執行以下 Python 語法來啟動工具：)*

```python
import Reference_easy_import
Reference_easy_import.run()
```

## 🎮 介面與操作說明 (UI & Usage Guide)

打開工具介面後，只需直覺地依序完成以下設定，按下 `Create Video Ref` 即可一鍵生成：

1. **Camera (選擇相機)**：指定參考影片要跟隨綁定的攝影機。
2. **Source Type & File (載入檔案)**：
   * 選擇 **Video (影片)**：選擇原檔，工具會自動在背後將其轉換為序列圖檔。
   * 選擇 **Image (序列圖)**：若已有轉好的圖檔，可直接選取。
3. **Target FPS (設定幀率)**：設定轉換序列圖的 FPS，系統預設會貼心地自動讀取並對齊當前 Maya 場景的 FPS。
4. **防誤觸保護 (鎖定 GRP)**：勾選「鎖定最外層 GRP」，可避免日後在調整動畫時不小心移動到參考結構。
5. **智慧播放起點**：生成完畢後，影片預設會直接從您點擊創建時「當下時間軸的畫格」作為第一幀開始播放。

## 🌟 核心功能與控制器操作 (Features & Control)

工具會為參考影片建立一套乾淨的層級架構，並賦予一個圓形的專屬控制器，讓您能完全掌控影片狀態：

* **直覺的畫面調整**：選取控制器，即可直接在畫面上自由移動、縮放參考影片的位置與大小。
* **Channel Box 專屬屬性**：
  * **`Offset` (整體平移)**：全域平移影片的時間點，輕鬆將影片動作與場景關鍵幀對齊。
  * **`Frame` (局部調整速度)**：支援在 Graph Editor 中對此屬性設定 Keyframe (關鍵幀)，方便針對特定動作局部調整播放速度或進行停格觀察。
* **攝影機完美跟隨**：無論鏡頭如何推軌或旋轉，參考畫面都會牢牢吸附在相機畫面上，是進行 Rotomation (動態轉描) 的絕佳利器。

## 🌟 核心功能 (Features)

本工具將以往需要點擊十幾次的重複性設定，濃縮為自動化流程：

### 1. 自動轉序列與匯入
依據當前動畫專案的 FPS 設定，自動幫你將影片轉換為序列圖檔並載入，大幅解決 Maya 直接讀取影片檔容易造成的卡頓問題。

### 2. 一鍵綁定攝影機
自動生成控制層級並吸附在選定的相機上，當鏡頭移動時，參考影片會完美跟隨視角。

### 3. 智慧時間軸對齊與偏移
匯入後，影片會自動從當前時間軸的第一幀開始播放。透過控制器上新增的 `offset` 屬性，可輕鬆全域平移影片的整體播放時間點。

### 4. 自訂局部播放速度
控制器上配置了專屬的 `frame` 屬性，支援直接在 Graph Editor 中設定關鍵幀 (Keyframe)，方便針對特定動作局部調整播放速度或停格觀察。

### 5. 優化 Layout 溝通流程
支援在同一個檔案內快速載入多個參考影片。在短時間切換多個鏡頭時，能確保動作連貫性，讓導演在 Layout 階段就能直觀確認畫面，減少溝通成本。

## 📝 系統相容性與開發備註 (Compatibility & Notes)

* **支援版本**：已於 Maya 2020 至 2025 版本進行測試 (Maya 2022 以上版本皆可穩定運行，2020 版本偶有不相容情況)。
* **開發背景**：本工具整合了過去在西基 (CGCG) 工作時的實務經驗與 Mandrake Animations 分享的概念，並透過 AI 輔助開發完成。致力於將繁瑣的技術設定自動化，讓動畫師能更快進入純粹的「動畫思考」狀態。
